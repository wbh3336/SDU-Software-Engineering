# 网上书店系统 - 前端页面

## 项目简介
本项目为网上书店前端界面，包含用户前台与管理员后台，
纯 HTML + CSS 实现，页面结构清晰，可直接对接后端接口。

## 功能模块

### 用户前台
- 首页（带背景图）
- 用户登录 / 注册
- 图书浏览页面
- 购物车页面
- 书店公告与留言

### 管理员后台（/admin）
- 管理员登录
- 控制台首页
- 进书管理
- 库存管理
- 订单管理
- 客户管理
- 账目统计
- 网站维护

## 技术栈
- HTML5
- CSS3
- 原生 JavaScript（简单交互）

## 说明
页面为静态前端版本，
交互逻辑与数据存储需对接后端接口实现。# SDU-Software-Engineering
